# RISC-V架构内联汇编

## 本章要求

1. 了解RISC-V指令集的基本知识。
2. 了解RISC-V架构的特点，以及如何在C语言中使用内联汇编。
3. 掌握C语言中常见的控制结构的汇编表示方法。
